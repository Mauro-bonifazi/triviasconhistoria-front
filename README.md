# Trivias con Historia – Frontend 🧠📜

Frontend de **Trivias con Historia**, una aplicación web educativa desarrollada con React, enfocada en aprendizaje interactivo, SEO y experiencia de usuario.

🔗 **Live Demo:**  https://triviasconhistoria-front.vercel.app
🔗 **Backend API:** https://github.com/Mauro-bonifazi/triviasconhistoria-back

---

## 🚀 Tech Stack

- React
- React Router
- Material UI
- Axios
- Context API (Auth)
- EmailJS

---

## ✨ Features

- Landing page con trivias destacadas
- Buscador por temática histórica
- Página de trivia con:
  - Reseña histórica
  - Preguntas dinámicas
  - Imágenes
  - Historial de resultados
- Panel de administración
- Formulario de contacto funcional
